---
title: Getting Started
description: Learn how to setup Astro Birb!
---


<Steps>
  <Step title="First Step">
    Run `/config` and select the staff roles & admin roles, you can learn about the permissions [here](/configuration/permissions.mdx) 
  </Step>
  <Step title="Second Step">
    Use `Config Menu` to find what module you want to use on the module there is a dropdown called `Module Toggle` select **True** to enabled that module.
  </Step>
  <Step title="Third Step">
    Now do the required actions for that module.
  </Step>
</Steps>
