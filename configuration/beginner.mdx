title: Getting Started
description: Learn how to setup Astro Birb!
---


<Steps>
  <Step title="How to enable Modules">
   1. Run /config.
   2. Click on Config Menu and then go on 'Modules'
   3. Use the Modules dropdown to select all the modules you wanna use. (You can scroll down)
   4. The Modules should now appear in your Config Menu.


  </Step>
  <Step title="Module Configuration">
  1. Press `Make a selection`.
  2. Click on the required options and fill out that option
  </Step>
  <Step title="All Done">
  It really isn't that hard.
  </Step>
</Steps>
